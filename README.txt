    [1]  What is CsTelnetd?
    ================================================================    
    
        CsTelnetd provides access to a Windows command-line 
        interface to clients connecting via Telnet over a network.

        CsTelnetd was developed as part of a school project
        regarding applications that implement threading.

    [5]  User Guide
    ================================================================
    
        *    Open the port the server will use, by default: 9999
        *    Run the executable cstelnetd.exe    
        *    Connect to the server via your favorite telnet client
        *    Execute desired commands
